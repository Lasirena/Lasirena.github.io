Demonbound - fully procedural dungeon crawler

Personal Project (2020)

In this latest solo Unity project, my goal was to tackle the creation of my own procedural algorithm for environment generation. The result is a Diablo-inspired ARPG prototype. Additional functionality may be added in the future.

Key features include:

  • Fully procedural and customizable 3D dungeon generation algorithm
  • Room population algorithm based on theme presets (implemented via ScriptableObjects)
  • New room presets and objects can be added to the population algorithm without writing any additional code
  • Smart object placement behaviour (e.g. avoidance of similar types, wall or central placement, etc.)
  • Depth-based dissolve shader (to allow the player to be visible behind walls)
  • Performant minimap solution for procedural levels

Space Surfer - pure C++ and OpenGL game

Personal Project (2020)

Built from the ground up in C++, this engine-less game focused solely around working with the modern standards of OpenGL.

Key features include:

  • Spline-based procedural path geometry
  • Custom implementation of the Blinn-Phong reflection model
  • Custom normal-mapping shader, and a geometry shader to visually debug mesh normals
  • Implementation of different light types (spot, point, directional)

Torrheim - a 2D narrative game

I worked on this Unity project with my colleague at the time when we were preparing some tutorials for Global Game Jam. During the jam, I wanted to focus on producing art assets.
